# Dataset Sources Reference

This document provides detailed information about all datasets referenced in the Manuscript benchmark, including access instructions, licensing, and citation information.

---

## Text Datasets

### 1. HC3 (Human ChatGPT Comparison Corpus)

**Description:** A comprehensive dataset comparing human-written answers with ChatGPT-generated responses across multiple domains.

**Access:**
- HuggingFace: https://huggingface.co/datasets/Hello-SimpleAI/HC3
- Direct download: `pip install datasets && python -c "from datasets import load_dataset; ds = load_dataset('Hello-SimpleAI/HC3')"`

**License:** Apache 2.0

**Size:** 37,000+ QA pairs

**Citation:**
```bibtex
@article{guo2023close,
  title={How Close is ChatGPT to Human Experts? Comparison Corpus, Evaluation, and Detection},
  author={Guo, Biyang and Zhang, Xin and Wang, Ziyuan and Jiang, Minqi and others},
  journal={arXiv preprint arXiv:2301.07597},
  year={2023}
}
```

### 2. Defactify-Text Dataset

**Description:** 58,000+ text samples combining authentic NYT articles with LLM-generated versions using GPT-4, Gemma, Mistral, Qwen, LLaMA, and Yi.

**Access:**
- Paper: https://arxiv.org/abs/2510.22874
- Dataset: Available through Defactify 4 competition (AAAI 2025)

**License:** Research use

**Citation:**
```bibtex
@inproceedings{defactify2025,
  title={A Comprehensive Dataset for Human vs. AI Generated Text Detection},
  booktitle={AAAI 2025 Workshop on Defactify},
  year={2025}
}
```

### 3. HATC-2025 (Human vs AI Text Corpus)

**Description:** Large-scale benchmark with 50,000+ samples for evaluating AI text detectors.

**Access:**
- Reference: https://hastewire.com/blog/ai-text-detection-benchmarks-2025-top-performance-metrics

### 4. LLMSciTxt Dataset

**Description:** Scientific papers written by humans vs. generated by ChatGPT, Gemini, and Llama-3.

**Access:**
- Paper: https://arxiv.org/html/2507.05157v1

### 5. Beemo Benchmark

**Description:** Compositions of human-written, machine-generated, expert-edited, and LLM-edited content.

**Access:**
- Website: https://toloka.ai/ai-detection-benchmark

## Image Datasets

### 1. MS COCOAI Dataset

**Description:** 96,000 image-caption pairs featuring real MS COCO images alongside AI-generated versions from SD3, SDXL, DALL-E 3, and Midjourney v6.

**Access:**
- Paper: https://arxiv.org/abs/2601.00553

**Structure:**
- Training: 42,000 samples
- Validation: 9,000 samples
- Test: 45,000 samples

**Citation:**
```bibtex
@article{mscocoai2025,
  title={A Comprehensive Dataset for Human vs. AI Generated Image Detection},
  journal={arXiv preprint arXiv:2601.00553},
  year={2025}
}
```

### 2. GenImage Dataset

**Description:** Million-scale dataset for AI-generated image detection featuring multiple generators.

**Access:**
- GitHub: https://github.com/GenImage-Dataset/GenImage

**Generators Included:**
- Midjourney
- Stable Diffusion
- ADM
- GLIDE
- Wukong
- VQDM
- BigGAN

**License:** Research use (NeurIPS 2023)

### 3. AIGIBench

**Description:** Benchmark including SD-XL, SD-3, DALL-E 3, Midjourney v6, FLUX, Imagen-3, and GLIDE.

**Access:**
- Paper: https://arxiv.org/html/2505.12335v1

### 4. Public Image Sources (for human samples)

**Unsplash:**
- Website: https://unsplash.com
- API: https://unsplash.com/developers
- License: Unsplash License (free for commercial use)

**Pexels:**
- Website: https://pexels.com
- API: https://www.pexels.com/api/
- License: Pexels License (free for commercial use)

**MS COCO:**
- Website: https://cocodataset.org
- License: CC BY 4.0 (annotations), various for images

## Audio Datasets

### 1. WaveFake Dataset

**Description:** 117,985 audio clips generated from LJSpeech using 7 different vocoder architectures.

**Access:**
- Zenodo: https://zenodo.org/record/5642694

**License:** CC BY 4.0

**Vocoders Included:**
- MelGAN
- Full-band MelGAN
- Multi-band MelGAN
- HiFi-GAN
- Parallel WaveGAN
- WaveGlow
- WaveRNN

**Citation:**
```bibtex
@inproceedings{frank2021wavefake,
  title={WaveFake: A Data Set to Facilitate Audio Deepfake Detection},
  author={Frank, Joel and Eisenhofer, Thorsten and others},
  booktitle={NeurIPS Datasets and Benchmarks Track},
  year={2021}
}
```

### 2. LJSpeech Dataset

**Description:** 13,100 short audio clips (~24 hours) of a single speaker reading passages from 7 non-fiction books.

**Access:**
- Website: https://keithito.com/LJ-Speech-Dataset/
- Direct download: https://data.keithito.com/data/speech/LJSpeech-1.1.tar.bz2

**License:** Public Domain

### 3. LibriSpeech

**Description:** 1000 hours of read English speech from audiobooks.

**Access:**
- Website: https://www.openslr.org/12/

**Subsets:**
- `train-clean-100`: 100 hours clean speech
- `train-clean-360`: 360 hours clean speech
- `test-clean`: Test set with clean speech

**License:** CC BY 4.0

### 4. ASVspoof Dataset

**Description:** Automatic Speaker Verification Spoofing and Countermeasures Challenge dataset.

**Access:**
- Website: https://www.asvspoof.org/
- Requires registration

**License:** Research use only

### 5. TIMIT-ElevenLabs

**Description:** Multi-speaker dataset for detecting ElevenLabs cloned voices.

**Reference:** https://arxiv.org/pdf/2307.07683

## Video Datasets

### 1. Deepfake-Eval-2024

**Description:** In-the-wild deepfakes from 2024, including Sora-generated content. 44+ hours of video, 56.5 hours of audio, 1,975 images.

**Access:**
- Paper: https://arxiv.org/html/2503.02857v1

**Key Finding:** Off-the-shelf detectors show 21.3% lower accuracy on Sora-like diffusion videos.

### 2. DeepfakeBench / DF40

**Description:** Comprehensive deepfake benchmark with 40 manipulation techniques.

**Access:**
- GitHub: https://github.com/SCLBD/DeepfakeBench

**License:** MIT

**Citation:**
```bibtex
@inproceedings{yan2023deepfakebench,
  title={DeepfakeBench: A Comprehensive Benchmark of Deepfake Detection},
  author={Yan, Zhiyuan and Zhang, Yong and others},
  booktitle={NeurIPS Datasets and Benchmarks Track},
  year={2023}
}
```

### 3. FaceForensics++

**Description:** 1.8M+ manipulated images and 1,000 original videos with 4 manipulation methods.

**Access:**
- GitHub: https://github.com/ondyari/FaceForensics
- Requires access request

**Methods:**
- DeepFakes
- Face2Face
- FaceSwap
- NeuralTextures

**Citation:**
```bibtex
@inproceedings{rossler2019faceforensics++,
  title={FaceForensics++: Learning to Detect Manipulated Facial Images},
  author={R{\"o}ssler, Andreas and Cozzolino, Davide and others},
  booktitle={ICCV},
  year={2019}
}
```

### 4. Microsoft Deepfake Dataset (2025)

**Description:** 50,000+ samples of real, AI-generated, and manipulated audio-visual content.

**Access:**
- Reference: https://www.biometricupdate.com/202507/new-microsoft-benchmark-for-evaluating-deepfake-detection-prioritizes-breadth

### 5. Kaggle DFD (Deep Fake Detection)

**Description:** Original deepfake detection dataset with 10,000+ videos.

**Access:**
- https://www.kaggle.com/datasets/sanikatiwarekar/deep-fake-detection-dfd-entire-original-dataset

### 6. Public Video Sources (for human samples)

**Pexels Videos:**
- Website: https://pexels.com/videos/
- API: https://www.pexels.com/api/documentation/#videos
- License: Pexels License (free for commercial use)

**Pixabay Videos:**

## Download Scripts

### Text Dataset (HC3)

```python
from datasets import load_dataset

# Load the HC3 dataset
dataset = load_dataset("Hello-SimpleAI/HC3")

# Access human and ChatGPT answers
for split in dataset:
    for item in dataset[split]:
        human_answer = item['human_answers']
        chatgpt_answer = item['chatgpt_answers']
```

### Audio Dataset (LJSpeech)

```bash
# Download LJSpeech
wget https://data.keithito.com/data/speech/LJSpeech-1.1.tar.bz2
tar -xjf LJSpeech-1.1.tar.bz2
```

### Audio Dataset (WaveFake)

```bash
# Download from Zenodo
wget https://zenodo.org/record/5642694/files/wavefake.zip
unzip wavefake.zip
```
